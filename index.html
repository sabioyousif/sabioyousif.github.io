U1: Programming

1.1: What Is CS?

  1. What is Computer Science?

    Computer Science is like learning about computers and how they work. It’s not just about the physical computer, but also the programs we use on them. This is important for all types of businesses, like banks or car companies, because they all use computer programs. In Computer Science, we learn to make these programs. We do this by creating step-by-step instructions, or algorithms, and then we design, build, test, and use them. It’s like building a Lego set, but with computer programs!

  2. What is Programming?

    Programming is when we give computers instructions to follow. Computers use a special code called binary that only understands "on" and "off" signals, but with programming languages, we can talk to computers in a way that’s easier for us to understand.

1.4: Introduction to Java

  1. WHAT IS JAVA?

    Java is a computer language used to make programs. It's special because it can run on any device that has Java, like computers, smartphones, or smart TVs.

  2. Classes and Methods

    In Java, every program starts with a class named after the file. Inside this class, there's a "main" method that lists the tasks of the program. To output something, we use: System.out.println("This is a String").

  3. Comments, Semicolons, and Whitespace

    Comments in code help us understand what's happening. They start with "//" for a single line, or "/* */" for multiple lines. Semicolons mark the end of a line of code. Whitespace, like empty lines or spaces, doesn't affect the code but helps with readability.

  4. Compiling, Classes, and Executables

    Java is a compiled language, which means it's converted into a class of byte code before it can run. This is done using "javac filename.java" in the Command Line, creating a "filename.class". This class can then be run using "java filename". This process is usually done automatically by the IDE (Integrated Development Environment) we use.

1.4.2: Variables in Java

  1. Importance and Uses of a Variable

    Variables are essential in all programming languages. They allow us to store data under a name, making it easier to use in the program. For instance, we can store the number 17 in the variable "userAge."

  2. Primitive Data Types in Java

    Java has three main primitive data types: int, double, and boolean. The 'int' type holds whole numbers, 'double' holds decimal numbers, and 'boolean' holds true or false values.

  3. Strings and Escape Sequences

    A String is an object within Java that holds a collection of characters. We use Strings to store words or sentences. An escape sequence allows us to perform special functions in Java without the computer misinterpreting what we're writing.

  4. Static Checking

    Java uses static typing, which means a program won't compile if a variable is assigned a value of an incorrect type. This helps catch bugs early.

  5. Naming Conventions

    When naming variables, Java uses camelCase. The first letter of the first word is lowercase, but the first letters of the remaining words are capitalized. Variable names should be relevant to the data you are storing and are case sensitive.
