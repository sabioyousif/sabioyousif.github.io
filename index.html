1.1: What Is CS?

  1. What is Computer Science?

    Computer Science is like learning about computers and how they work. It’s not just about the physical computer, but also the programs we use on them. This is important for all types of businesses, like banks or car companies, because they all use computer programs. In Computer Science, we learn to make these programs. We do this by creating step-by-step instructions, or algorithms, and then we design, build, test, and use them. It’s like building a Lego set, but with computer programs!

  2. What is Programming?

    Programming is when we give computers instructions to follow. Computers use a special code called binary that only understands "on" and "off" signals, but with programming languages, we can talk to computers in a way that’s easier for us to understand.

1.4: Introduction to Java

  1. WHAT IS JAVA?

    Java is a computer language used to make programs. It's special because it can run on any device that has Java, like computers, smartphones, or smart TVs.

  2. Classes and Methods

    In Java, every program starts with a class named after the file. Inside this class, there's a "main" method that lists the tasks of the program. To output something, we use: System.out.println("This is a String").

  3. Comments, Semicolons, and Whitespace

    Comments in code help us understand what's happening. They start with "//" for a single line, or "/* */" for multiple lines. Semicolons mark the end of a line of code. Whitespace, like empty lines or spaces, doesn't affect the code but helps with readability.

  4. Compiling, Classes, and Executables

    Java is a compiled language, which means it's converted into a class of byte code before it can run. This is done using "javac filename.java" in the Command Line, creating a "filename.class". This class can then be run using "java filename". This process is usually done automatically by the IDE (Integrated Development Environment) we use.

1.4.2: Variables in Java

  1. Importance and Uses of a Variable

    Variables are essential in all programming languages. They allow us to store data under a name, making it easier to use in the program. For instance, we can store the number 17 in the variable "userAge."

  2. Primitive Data Types in Java

    Java has three main primitive data types: int, double, and boolean. The 'int' type holds whole numbers, 'double' holds decimal numbers, and 'boolean' holds true or false values.

  3. Strings and Escape Sequences

    A String is an object within Java that holds a collection of characters. We use Strings to store words or sentences. An escape sequence allows us to perform special functions in Java without the computer misinterpreting what we're writing.

  4. Static Checking

    Java uses static typing, which means a program won't compile if a variable is assigned a value of an incorrect type. This helps catch bugs early.

  5. Naming Conventions

    When naming variables, Java uses camelCase. The first letter of the first word is lowercase, but the first letters of the remaining words are capitalized. Variable names should be relevant to the data you are storing and are case sensitive.

U3: Sorting

Sorting Algorithms

  1. Understanding Sorting Algorithms

    Sorting algorithms are techniques used to organize elements in a list or array in a certain order, like from smallest to largest. They work by comparing elements and swapping their places to achieve the desired order. Examples include QuickSort, MergeSort, and BubbleSort, each with different efficiencies and uses.

  2. The Importance of Sorting

    Sorting is crucial in Computer Science as it makes data retrieval faster and improves algorithm efficiency. It's key in data analysis for tasks like finding averages or patterns. Sorting also forms the basis of more complex algorithms, making it a fundamental concept in the field. Additionally, sorting helps us understand the limits of computation.

  3. Examples of Sorting Algorithms

    There are many sorting algorithms, each with its own strengths and weaknesses. For example, BubbleSort is simple but not very efficient for large lists, while QuickSort and MergeSort are more complex but can handle larger lists more efficiently.

Time Complexity & Big O Notation

  1. Understanding Time Complexity

    Time complexity measures how an algorithm's runtime grows with the size of the input, helping us gauge its efficiency. Big O Notation is a way to express the worst-case scenario of an algorithm's time or space complexity. It helps us understand how an algorithm's performance changes with larger inputs. Examples include O(n) for linear time and O(log n) for logarithmic time.

  2. Recursive Programming

    A recursive program solves a problem by breaking it down into smaller versions of the same problem until it reaches a base case that stops the recursion. From this base case, the program works its way back up to the final answer. This is different from an iterative program that uses loops to repeat operations until a condition is met. While recursive solutions might be faster, they might use more memory due to the function calls.

U4: Topics in CS

4.1 - Environmental

  1. Understanding E-waste

    E-waste is electronic products that are no longer wanted or functional. This includes items like computers, smartphones, and TVs.

  2. The Impact of E-waste on the Environment

    E-waste contains harmful substances like lead, mercury, and cadmium. If not disposed of correctly, these can pollute the air, soil, and water.

  3. Global E-waste Production

    Each year, over 50 million tonnes of e-waste are produced worldwide. Unfortunately, only a small portion of this is recycled properly.

  4. Managing E-waste Responsibly

    Recycling e-waste can recover valuable materials and lessen its environmental impact. It's important to dispose of e-waste correctly to handle hazardous substances safely.

  5. E-waste Recycling Programs

    Many countries have set up e-waste recycling programs and laws to promote the responsible disposal and recycling of electronic devices.

Ergonomics in Computer Science

  1. What is Ergonomics?

    Ergonomics is about designing equipment and devices to fit the human body and its movements. In Computer Science, it's about setting up a workspace and using tools that minimize strain and injury while maximizing productivity and comfort.

  2. Maintaining Good Posture

    Keeping a good posture can help reduce strain on your back and neck.

  3. Supporting Your Wrists

    Using wrist supports and ergonomic keyboards can help prevent wrist injuries.

  4. Choosing a Comfortable Chair

    Investing in a comfortable chair with good lumbar support can make a big difference.

  5. Positioning Your Monitor

    Adjusting the height and distance of your monitor can help reduce eye strain.
